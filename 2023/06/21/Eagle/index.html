一、概述

Kafka-eagle是一个为了简化开发者和服务工程师维护Kafka集群工作的监控管理工具。这个管理工具可以很容易地发现分布在集群中的哪些topic分布不均匀，或者是分区在整个集群分布不均匀的的情况。它支持管理多个集群、选择副本、副本重新分配以及创建Topic。同时，这个管理工具还可以快速浏览整个集群。

Kafka Eagle所包含的功能有数据面板、数据大屏、主题列表、消费者组、集群详情、监控中心、告警策略、系统管理等功能

二、安装部署

1、环境需求

需要安装JDK、ZooKeeper和Kafka。

2、上传安装包并解压

下载地址：http://download.kafka-eagle.org/
下载完成后上传到/export/server开始解压 tar -zxvf kafka-eagle-web-2.0.2-bin.tar.gz -C ../servers

cd /export/servers/ kafka-eagle-web-2.0.2/tar -zxvf kafka-eagle-web-2.0.2bin.tar.gz`

3、修改环境变量

JAVA_HOME 和 KE_HOME

vi /etc/profile

export JAVA_HOME=/usr/java/jdk1.8

export PATH=PATH:JAVA_HOME/bin

export KE_HOME=/export/server/kafka-eagle

export PATH=PATH:KE_HOME/bin

4、 配置 KafkaEagle

cd /export/server/kafka-eagle/conf

vi system-config.properties

l 需要更改的地方：

kafka.eagle.zk.cluster.alias=cluster1

cluster1.zk.list=node1:2181,node2:2181,node3:2181

cluster1.kafka.eagle.broker.size=3

kafka.eagle.url=jdbc:sqlite:/export/data/db/ke.db

5、启动前需要手动创建/export/data/db目录

mkdir /export/data/db

6、启动Eagle

/export/server/kafka-eagle/bin/ke.sh start
